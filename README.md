#  OpenCV Computer Vision Project

A Python-based Computer Vision project built using **OpenCV**. This project demonstrates image processing, video processing, webcam access,and basic computer vision operations.

---

## 📌 Features

- 📷 Read and display images
- 🎥 Read and play video files
- 📹 Access webcam in real time
- 💾 Save images and videos
- 🌈 Convert images to grayscale
- ⚡ Edge detection using Canny
- 🧹 Image smoothing and blurring
- 🔲 Thresholding
- 😊 Face detection using Haar Cascades


---

## 🛠️ Technologies Used

- Python 3.x
- OpenCV
- NumPy
